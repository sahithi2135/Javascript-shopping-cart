# Javascript-shopping-cart
##Project Overview
Lumen is a responsive and user-friendly e-commerce web application designed to provide a seamless online shopping experience. The platform enables users to browse products, search items, filter products by category, add products to a shopping cart, manage cart items, and place orders through an intuitive and modern interface. The application focuses on delivering a smooth user experience with dynamic product rendering, real-time cart updates, and responsive design across different devices.

Built using HTML, CSS, and JavaScript, the project integrates product data through APIs and utilizes Local Storage for cart persistence, allowing users to retain their shopping selections throughout their session. The application follows modern web development practices and demonstrates frontend development skills, DOM manipulation, API integration, responsive UI design, and client-side data management.

##Key Features
Product Catalog with Dynamic Rendering
Product Search Functionality
Category-Based Product Filtering
Shopping Cart Management
Add to Cart and Remove from Cart
Real-Time Cart Quantity Updates
Order Summary and Checkout Flow
Local Storage Integration
Responsive Design for Mobile, Tablet, and Desktop
Interactive User Interface with Modern Styling
